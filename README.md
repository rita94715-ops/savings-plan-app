# 💰 存錢計畫 - Savings Plan App

一個可添加到 iPhone 主畫面的 PWA（Progressive Web App）存錢計畫應用。

## ✨ 功能特性

- 📱 **PWA 應用** - 可以像原生應用一樣添加到 iPhone 主畫面
- 💾 **本地數據存儲** - 數據保存在設備本地，無需上傳服務器
- 🎨 **自訂背景顏色** - 8 種預設顏色可選
- 📝 **自訂名稱** - 每個計畫最多 20 個字
- 🎯 **目標設定** - 自由設定存錢目標金額
- 📊 **進度追蹤** - 實時百分比顯示
- ⚡ **快速調整** - ± 按鈕和快捷按鈕快速增減金額
- 🔓 **離線使用** - Service Worker 緩存，斷網也能用

## 🚀 快速開始

### 安裝依賴
```bash
npm install
```

### 本地開發
```bash
npm run dev
```
應用會在 `http://localhost:5173` 打開

### 生產構建
```bash
npm build
```

## 📱 添加到 iPhone 主畫面

### 使用 Safari 瀏覽器

1. 在 iPhone 上打開 Safari 瀏覽器
2. 輸入應用的網址
3. 點擊底部的分享按鈕（方形中有向上箭頭）
4. 向下滑動，點擊「加入主畫面」
5. 輸入應用名稱（可選），點擊「加入」

應用就會出現在你的主畫面上，點擊即可打開！

## 💾 數據存儲

所有計畫數據都存儲在瀏覽器的 `localStorage` 中：
- 即使關閉應用，數據也會被保留
- 清除瀏覽器數據時會丟失
- 不會上傳到任何服務器

## 🛠️ 技術棧

- **Vue.js 3** - 用戶界面框架

- **PWA** - 漸進式網頁應用

## 📝 使��說明

### 創建計畫
1. 點擊「+ 新增計畫」按鈕
2. 填寫計畫名稱和目標金額
3. 選擇喜歡的背景顏色
4. 點擊「保存」

### 增加金額
- 使用「➕」和「➖」按鈕調整
- 或輸入金額後點擊「➕」
- 或直接點擊快捷按鈕（+100、+500、+1000）

### 編輯計畫
1. 點擊計畫卡片右上角的 ✏️ 按鈕
2. 修改信息
3. 點擊「保存」

### 刪除計畫
1. 點擊計畫卡片右上角的 🗑️ 按鈕
2. 確認刪除
